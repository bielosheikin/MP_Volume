WHAT IS THIS?
This is a standalone executable with EVERYTHING built-in:
- Python runtime
- PyQt5 GUI framework  
- Matplotlib plotting library
- NumPy numerical computing
- All application code

NO INSTALLATION REQUIRED. NO PYTHON NEEDED. JUST RUN IT!

================================================================================
HOW TO USE:
================================================================================

1. Download MP_Volume.exe
2. Double-click to run
3. That's it!

The first launch may take 5-10 seconds as the application extracts itself
to a temporary location. Subsequent launches are faster.

================================================================================
SYSTEM REQUIREMENTS:
================================================================================

- Windows 10 or Windows 11 (64-bit)
- 4 GB RAM minimum
- 100 MB free disk space (for temporary extraction)
- Display: 1280x720 minimum resolution

================================================================================
ANTIVIRUS WARNING:
================================================================================

Some antivirus programs may flag PyInstaller executables as suspicious.
This is a FALSE POSITIVE. The file is safe.

If blocked:
- Add an exception in your antivirus
- Right-click → Properties → Unblock (if present)
- Run as Administrator (if necessary)

================================================================================
HOW IT WORKS:
================================================================================

When you run MP_Volume.exe:
1. It extracts necessary files to: C:\Users\YourName\AppData\Local\Temp\_MEI...
2. Runs from that temporary location
3. Cleans up when you close the application

This happens automatically and is transparent to you.
