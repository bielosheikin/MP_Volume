# - Matplotlib 3.8.4: Matches NumPy compatibility
#
# Breaking changes to watch:
# - NumPy 2.0+ has API changes; current code uses 1.26.x
# - PyQt6 is a major rewrite; migration would require GUI updates
# - Matplotlib 3.9+ may have new default styles
#
# ----------------------------------------------------------------------------
# Installation Troubleshooting
# ----------------------------------------------------------------------------
#
# Issue: PyQt5 installation fails
# Solution: Install Qt5 system libraries first or use binary wheel
#   - Windows: Should work automatically with pip
#   - Linux: sudo apt-get install python3-pyqt5
#   - macOS: brew install pyqt5
#
# Issue: NumPy installation fails on Apple Silicon
# Solution: Use conda or ensure Xcode command line tools installed
#   - conda install numpy
#   - xcode-select --install
#
# Issue: PyInstaller executable doesn't run
# Solution: Check antivirus settings, try building with --debug flag
#
# Issue: Matplotlib backend errors
# Solution: Set backend explicitly in code or use TkAgg/Qt5Agg
#
# ----------------------------------------------------------------------------
# Building Executables
# ----------------------------------------------------------------------------
#
# After installing requirements, build with:
#   Windows: build_exe.bat
#   Linux/Mac: ./build_exe.sh
#   Manual: pyinstaller MP_Volume.spec --clean
#
# Expected build size: ~70 MB (single file executable)
# Build time: 5-10 minutes on first build
